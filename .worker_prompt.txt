## Design Aesthetic Direction

**Aesthetic:** Editorial Warm — think "Airbnb × Substack × cult D2C brand"
- Warm off-white backgrounds (like Substack/Linear light)
- Serif italic display type (Fraunces) paired with crisp sans body (Inter Tight)
- Orange `#FF4716` as the dominant accent, used confidently not timidly
- Soft-rounded corners (radius 14-16px default, 24px on big cards)
- Subtle warm shadows (`rgba(255, 71, 22, 0.08)` tint, never pure black shadows)
- Grain/noise overlay on hero sections for depth
- Custom icons matching the flower-petal aesthetic (rounded ends, organic curves)
- NO generic lucide default icons — we build a matching set

**Light theme (new DEFAULT):**
```
--bg:          #FAF7F2   (warm off-white, like paper)
--bg-elevated: #FFFFFF   (pure white for cards that pop)
--border:      #EEE8DD   (warm cream border)
--text:        #0B0A08   (near-black, warm)
--muted:       #6B6558   (warm stone)
--primary:     #FF4716   (brand orange)
--primary-fg:  #FFFFFF
--success:     #3E7C4B
--warning:     #C88A10
--destructive: #C8301A
```

**Dark theme (alt, via toggle):**

**Typography:**
- Display (headlines, brand): **Fraunces** 600, italic — gives that editorial feel
- Body UI: **Inter Tight** 400/500/600 (tighter than regular Inter, more modern)
- Mono (IDs, timestamps): **JetBrains Mono** 400

Install the fonts into admin via Google Fonts CDN, into mobile via `@expo-google-fonts/fraunces` + `@expo-google-fonts/inter-tight`.
