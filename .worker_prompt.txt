## Specific requirements for the scaffold

### server/package.json
- Name: `indigen-server`
- Scripts: `dev` (tsx watch src/index.ts), `start` (tsx src/index.ts), `build` (tsc), `migrate` (runs SQL files in db/migrations against $DATABASE_URL)
- Dependencies: `express`, `cors`, `helmet`, `pg`, `jsonwebtoken`, `bcrypt`, `dotenv`, `zod`, `pino`, `pino-http`, `ioredis`, `bullmq`, `node-cron`, `axios`, `cashfree-pg`
- DevDependencies: `typescript`, `tsx`, `@types/node`, `@types/express`, `@types/cors`, `@types/pg`, `@types/jsonwebtoken`, `@types/bcrypt`

### server/tsconfig.json
Standard Node strict TS config (target ES2022, module commonjs, strict true, esModuleInterop true, outDir dist).

### server/src/index.ts
Create a minimal but real Express app that:
- Loads dotenv
- Mounts cors + helmet + json body parser + pino-http logger
- Registers a `/health` endpoint returning `{ status: 'ok', uptime, version }`
- Listens on `process.env.PORT || 3001`
- Logs "Indi-gen API server on :PORT" on boot
- Has placeholder comments where future routes will mount

### scraper-service/app/main.py
FastAPI app with:
- POST `/scrape/search` (keywords: list[str], limit: int) вЂ” returns stub `{ "status": "not_implemented" }`
- GET `/scrape/profile/{public_id}` вЂ” returns stub
- GET `/health` вЂ” returns `{ "status": "ok" }`
- Runs on port 8000
